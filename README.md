# Timelines

Tags: jQuery, javascript, php, css
Requires at least: 3.6.0
Tested up to: 6.9.1
License: GPL2

## Description

Adds a timeline function as a CPT to inserted into a post or a page using shortcode with vertical & horizontal options.

## Tested on 
* Firefox 
* Safari
* Chrome
* Opera
* MS Edge

## Website 
http://www.phildesigns.com/

## Prerequisite
* Requires ACF Pro

## Installation 
1. Upload ‘timelines’ to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "Timelines" custom post type and create timelines there
4. Use the shortcode to embed the time line on your posts and pages, choose between vertical and horizontal layouts
